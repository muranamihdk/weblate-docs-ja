Weblate について¶

プロジェクトの目標¶

Git と緊密に連携し、多くのファイルフォーマットをサポートした、翻訳者の貢献を容易にするウェブベースの翻訳ツール。

翻訳結果はソースコードとして同一リポジトリに保存されること、また、翻訳プロセスは開発にきっちりと追従すること。

深刻な競合を解決する手段を提供する計画はない。これらは主に Git によって処理されるべきであるため。

プロジェクトの名前¶

このプロジェクトの名前は "web" と "translate" という言葉の組み合わせとして名付けられました。

プロジェクトのウェブサイト¶

プロジェクトのウェブサイトは <https://weblate.org/> にあります。デモ・サーバも <https://demo.weblate.org/> にあります。ホスティング・サービスを <https://hosted.weblate.org/> で提供しています。このドキュメントは <https://docs.weblate.org/> で閲覧可能です。
